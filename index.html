Outlook (New)

File → Settings → Account → Signature → click New → name it → click in the signature box → Ctrl+V to paste → Save

Outlook (Classic)

File → Options → Mail → Signatures → click New → name it → click in the signature box → Ctrl+V to paste → Save

Outlook on the Web (outlook.com / outlook.office.com)

Click the gear icon (Settings) → Mail → Compose and reply → under "Email signature" click in the box → Ctrl+V to paste → Save
